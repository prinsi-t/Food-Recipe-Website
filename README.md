# 🍲 Recipe App

A simple and responsive recipe web app where users can explore different recipes with detailed ingredients and step-by-step cooking instructions.

## 🚀 Features

- Browse all recipes in one place  
- View ingredients for each recipe  
- Step-by-step cooking instructions  
- Recipes fetched from external API + stored in MongoDB  
- Combination of dynamic API data and custom database recipes  

## 🛠️ Tech Stack

- Node.js  
- Express.js  
- MongoDB  
- Tailwind CSS  
- JavaScript  

## 📦 Run Locally

```bash
npm install
npm run dev
